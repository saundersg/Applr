For this weeks analysis, we worked with a friendly competition data sets from Kaggle.com about predicting final sales prices for houses, based on 79 available explanatory variables categorising and describing most aspects of residential houses in Ames, Iowa.

I based some of this lm model on what we had already done in class and the other x variables came from the pairs plots while looking for any kind of notable slope. Data Validation

This model validation has an R-square of about 0.83, meaning this regression model can be expected to explain 80% of its variability. It should be noted that a lot of interactions make up this model, making its interpretability harder.

set.seed(121)
num_rows <- 1000 # 1460 total
keep <- sample(1:nrow(HousesData), num_rows)

mytrain <- HousesData[keep, ] #Use this in the lm(..., data=mytrain) it is like "rbdata"

mytest <- HousesData[-keep, ] #Use this in the predict(..., newdata=mytest) it is like "rbdata2"

lm.con <- lm(data=mytrain, SalePrice ~ RichNbrhd + TotalSF:RichNbrhd + GarageArea + GrLivArea + TotalSF:GrLivArea + RichNbrhd:TotRmsAbvGrd + as.factor(KitchenQual2):TotalSF + OverallQual + OverallQual:TotalSF)

# Validation set-up
yh_con <- predict(lm.con, newdata=mytest)
ybar <- mean(mytest$SalePrice)

SSTO <- sum( (mytest$SalePrice - ybar)^2 )
SSE_con <- sum( (mytest$SalePrice - yh_con)^2 )

rs_con <- 1 - SSE_con/SSTO

n <- nrow(mytest)
p_con <- length(lm.con)

rsa_con <- 1 - (n-1)/(n-p_con)*SSE_con/SSTO
my_output_table <- data.frame(Model = "Model Validation",
                                 `Original R2` = summary(lm.con)$r.squared, 
                                 `Orig. Adj. R-squared` = summary(lm.con)$adj.r.squared,
                                 `Validation R-squared` = rs_con, 
                                 `Validation Adj. R^2` = rsa_con)

colnames(my_output_table) <- c("Model", "Original $R^2$", "Original Adj. $R^2$", "Validation $R^2$", "Validation Adj. $R^2$")
